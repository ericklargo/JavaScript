JavaScript es un lenguaje de programación ligero e interpretado que maneja funciones de primera clase, es decir, las funciones pueden tratarse como variables u objetos comunes.

Esto permite asignar funciones a variables de la misma manera que en otros lenguajes de programación. Una característica importante es que las variables pueden almacenar funciones y ejecutarlas posteriormente.

Paradigmas de Programación Soportados

JavaScript soporta múltiples paradigmas de programación:

* Imperativo

* Funcional

* Orientado a Objetos

Paradigma Utilizado
- Orientado a Objetos

En este ejemplo definimos una clase que encapsula estado y comportamiento mediante objetos, haciendo uso explicito del encapsulamiento.
